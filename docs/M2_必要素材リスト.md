# Milestone 2 着手に必要な素材・情報リスト

---

**作成日：** 2026年5月25日
**対象フェーズ：** Phase 1 / Milestone 2（認証・決済・チケット発行）

---

## ご提供いただきたい情報

Milestone 2 の開発着手に向けて、以下の情報・素材をご提供ください。

---

### 1. Stripe アカウント情報（必須）

決済機能の実装に必要です。

| 項目 | 内容 |
|------|------|
| Stripe 公開可能キー | `pk_live_` または `pk_test_` から始まるキー |
| Stripe シークレットキー | `sk_live_` または `sk_test_` から始まるキー |
| Stripe Webhook シークレット | Webhook エンドポイント設定後に発行 |

> **備考：**
> - テスト環境では `pk_test_` / `sk_test_` キーを使用します
> - 本番キーは本公開直前に切り替えます
> - Stripe アカウントをお持ちでない場合は [stripe.com](https://stripe.com) より作成をお願いします
> - シークレットキーは安全な方法（暗号化チャット等）でご共有ください

---

### 2. Stripe 商品・価格設定（必須）

チケット種別ごとに Stripe 上で商品登録が必要です。

| 項目 | 内容 |
|------|------|
| チケット種別名 | 例：2ショットチェキ券、サイン券 等 |
| 各チケットの価格（円） | 例：3,000円、5,000円 等 |
| 在庫数 | 各種別ごとの販売上限枚数 |

> **備考：** こちらで Stripe ダッシュボードへの商品登録も代行可能です。上記情報をご提供いただければ対応いたします。

---

### 3. デザイン・画面仕様の確認（必須）

実装に際し、以下のデザインURLの最終確認をお願いします。

| アプリ | URL |
|--------|-----|
| ファン向けアプリ | https://idol-deploy.vercel.app/ |
| スタッフ向けアプリ | https://mobile-app-code.vercel.app/admin/master |
| 運営管理画面（PC） | https://mobile-app-code.vercel.app/admin-pc/dashboard |

確認事項：
- [ ] 画面デザインに変更予定がないか
- [ ] カラー・フォント等のブランドガイドラインがあれば共有
- [ ] 各画面の遷移フローに追加・変更がないか

---

### 4. メール認証設定（推奨）

Supabase Auth のメール確認フロー用に、以下をご確認ください。

| 項目 | 内容 |
|------|------|
| 会員登録時のメール確認 | 必要 / 不要 |
| 送信元メールアドレス | 例：noreply@passi.jp |
| サービス表示名 | 例：PassI |

> **備考：** 独自ドメインのメール送信が必要な場合は、SendGrid 等の連携が別途必要です。

---

### 5. 本番環境ドメイン（参考情報）

テストリリースに向けて、ドメインが決まっていれば事前にご共有ください。

| アプリ | 想定URL例 |
|--------|-----------|
| ファン向け | https://passi.jp |
| スタッフ向け | https://staff.passi.jp |
| 管理画面 | https://admin.passi.jp |

> **備考：** 未定の場合は Vercel の自動付与ドメインで暫定運用します。
