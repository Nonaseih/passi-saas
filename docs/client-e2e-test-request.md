## 確認手順

### 1. ファンアカウントでログイン
アプリにファンとしてログイン（またはテスト用アカウントを新規作成）してください。

### 2. チケット購入画面へ
イベント一覧からチケット種別を選択し、「Stripeで決済する」ボタンをタップしてください。

### 3. テスト用カード情報を入力
Stripe の決済画面が開いたら、以下のテスト用カード番号を入力してください。

| 項目 | 入力値 |
|------|--------|
| カード番号 | `4242 4242 4242 4242` |
| 有効期限 | `12/34`（将来の日付であれば何でも可） |
| CVC | `123`（任意の3桁） |
| 名前 | 任意 |

「支払う」をクリックしてください。

### 4. 購入完了後の確認
アプリの **「マイチケット」** 画面に戻り、以下をご確認ください。

- [ ] 購入したチケットが表示される
- [ ] チケットをタップするとQRコードが表示される
- [ ] ステータスが「未使用」になっている
