# Stripe Webhook Secret — Setup Request

Hi [Client name],

As we move into the payment integration phase of PassI, we need one more item from your Stripe account to complete the setup: a **webhook signing secret**.

This is a short one-time step — about 3 minutes in your Stripe Dashboard.

---

## What we need

A webhook signing secret (`whsec_...`) generated by Stripe.  
This lets our server verify that payment confirmation events actually came from Stripe and weren't faked.

---

## Steps to get it

1. Log into your Stripe Dashboard at **dashboard.stripe.com**
2. In the left sidebar, go to **Developers → Webhooks**
3. Click **"Add endpoint"**
4. In the **Endpoint URL** field, enter:
   ```
   https://isevhcauafjdxspyqfvu.supabase.co/functions/v1/stripe-webhook
   ```
5. Under **"Select events to listen to"**, add these three events:
   - `checkout.session.completed`
   - `payment_intent.payment_failed`
   - `account.updated`
6. Click **"Add endpoint"** to save
7. On the next screen, click **"Reveal"** under **Signing secret**
8. Copy the value — it starts with `whsec_`

Please send us that `whsec_...` value and we'll handle the rest.
